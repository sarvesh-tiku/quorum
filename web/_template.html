How the gate works

01 — DECIDER

One single-threaded brain

Claude Opus 4.8 owns every decision (Cognition's "don't build multi-agents"). For each step it emits an action and a checkable claim about the post-state — turning "trust me" into a falsifiable assertion.

02 — CONSENSUS GATE

A jury of read-only jurors

Many parallel Claude Haiku 4.5 jurors see only the state + action + claim — never the brain's reasoning (kills agreement bias). Each runs its own query. red-flag drops junk votes; first-to-ahead-by-K decides — cheap when obvious, more jurors only when uncertain.

03 — RECOVERY

Rewind & re-plan

On FAIL the jurors localize the root cause, the world rolls back to the last good snapshot, and the brain re-plans with the juror evidence in context. The bad commit never compounds.
